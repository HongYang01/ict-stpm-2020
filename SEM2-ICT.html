1. Introduction to Programing

Define Computer Program
  • A computer program is a series of instructions that directs a computer to perform tasks
Generation of Programming Language

Generation of Programming Language

1. Low-level
  • is a programming language that provides little or no abstraction from a computer's microprocessor.
2. High-level
  • is a programming language that is more abstract, easier to use and more portable across platforms.

Type of Language Translator

1. Assembler
  • convert assembly language into machine language
2. Interpreter
  • interpret, convert to machine language & execute line by line
2. Compiler
  • convert entire source code into machine language & execute

Arithmetic operators

Relational operators

>= Larger equal than

Logical operators
&& AND
|| OR
! NOT
Assignment Operators
= assign
+= adds right operand to the left operand and assign the result to left
-= subtracts right operand from the left operand and assign the result to left operand
*= multiply left operand with the right operand and assign the result to left operand
/= divides left operand with the right operand and assign the result to left operand
%= calculate modulus using two operands and assign the result to left operand
Special operators
